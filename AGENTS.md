# Package manager: bun

This project uses **bun**, not npm. Use `bun install`, `bun add <pkg>`, `bun remove <pkg>`, `bun run <script>` (e.g. `bun run dev`, `bun run build`, `bun run lint`). Never run `npm install` or `npm add`.
